Project Name: DigiTools Platform

DigiTools is a fast and clean web application where users can find premium digital products. I built it to practice real-world frontend development with a professional design.

Technology:-
Frontend Library: React.js (Vite)
Styling         : Tailwind CSS & DaisyUI
Icons           : Lucide React & own Assets
Alert           : React-Toastify

Features about this project:-

1.Navbar: When scrolling then the navbar is sticky which looking so impressive.When its responsive then dropdown menu showing which can makes user friendly.

2.Cart: We can easily add or remove products and the cart count instantly upgraded.

3.Alert: when we added product or remove or checkout products then instantly pop-up an toster alert which is the most interesting feeling for using this web app.
